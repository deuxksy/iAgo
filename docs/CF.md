# **프로젝트 이아고(Iago): 알라딘 만화 카테고리 심층 분석 모바일 웹 구축**

## **1 서론 (Introduction)**

디지털 콘텐츠 소비의 급격한 변화 속에서도 출판 만화(Comics) 시장은 소장 가치와 팬덤을 중심으로 견고한 성장세를 유지하고 있다. 국내 최대 온라인 서점 알라딘(Aladin.co.kr)은 방대한 만화 데이터베이스를 보유하고 있으나, 마케팅과 트렌드 분석에 필수적인 '구매자 인구통계학적 분포(Demographics)'와 '세부 평점 분포(Rating Distribution)' 데이터는 API로 제공되지 않고 웹사이트 깊숙이 숨겨져 있다.1  
본 보고서는 알라딘의 데이터를 수집·분석하여 사용자에게 직관적인 만화 선택 가이드를 제공하는 모바일 웹 애플리케이션 \*\*'이아고(Iago)'\*\*를 구축하는 기술 명세서이다.

### **1.1 프로젝트 추진 배경 및 동기 (Project Motivation)**

본 프로젝트는 개발자의 실질적인 니즈와 기술적 학습 목표를 충족하기 위해 추진된다.

1. **만화 선택의 의사결정 지원 (Overcoming Decision Fatigue):** 만화책 애호가로서, 방대한 신간과 명작들 사이에서 무엇을 읽을지 고민하는 시간을 줄이고자 한다. "그냥 아무거나 보고 싶을 때"에도 실패하지 않는 선택을 할 수 있도록, 특정 세대나 성별이 검증한 데이터를 기반으로 **'믿고 보는 추천'** 혹은 **'우연한 발견'**을 돕는 도구를 직접 제작한다.  
2. **바이브 코딩(Vibe Coding) 방법론 체득:** 단순한 코딩 실습을 넘어, AI를 주도적인 파트너로 활용하는 '바이브 코딩' 방법론을 실전에 적용한다. 기획부터 배포까지의 전 과정에서 AI와 협업하며 개발 속도와 생산성을 극대화하는 경험을 쌓고, 미래지향적인 개발 역량을 확보하는 것을 목표로 한다.

## **2 프로젝트 관리 및 기획 (Project Management & Planning)**

### **2.1 프로젝트 개요 및 목표 정의**

프로젝트 '이아고'는 단순한 베스트셀러 목록을 넘어, 데이터에 숨겨진 맥락을 시각화하여 사용자의 선택을 돕는다.

1. **데이터의 정량화:** 텍스트와 그래프 이미지로만 존재하는 구매자 분포 및 평점 데이터를 수치 데이터(Numeric Data)로 변환한다.  
2. **모바일 퍼스트(Mobile-First) 경험:** 이동 중이나 서점에서 바로 확인할 수 있도록 스마트폰 환경에 최적화된 인터페이스를 제공한다.  
3. **직관적 탐색 (Serendipity & Insight):** "30대 남성이 열광하는 숨은 명작" 등을 필터링하여, 사용자가 고민 없이 콘텐츠를 선택할 수 있게 한다.  
4. **AI 주도 개발 학습:** 바이브 코딩을 통해 '자연어 기획 -> AI 코드 생성 -> 인간의 검증'으로 이어지는 워크플로우를 완벽하게 숙달한다.

### **2.2 바이브 코딩(Vibe Coding) 방법론의 적용**

'바이브 코딩'은 개발자가 AI에게 의도(Vibe)를 전달하고 결과물을 조율하는 감독 역할을 수행하는 방식이다.3 본 프로젝트는 이 방법론을 학습하기 위한 파일럿 프로젝트의 성격을 가진다.

#### **2.2.1 역할별 바이브 코딩 워크플로우**

| 역할 | 전통적 방식 | 바이브 코딩 방식 (학습 포인트) | AI 협업 포인트 |
| :---- | :---- | :---- | :---- |
| **PM** | 상세 기획서 작성 | **PRD & Context 파일 작성** | 내 의도를 AI가 완벽히 이해하도록 구조화된 텍스트(Markdown)로 명령하는 법 훈련 5 |
| **Data** | 로직 직접 구현 | **로직 설계 및 에러 핸들링** | Playwright의 복잡한 선택자 및 정규식을 AI에게 맡기고, 데이터 무결성 검증에 집중 6 |
| **Front** | HTML/CSS 마크업 | **컴포넌트 조립 및 디자인** | "깔끔하고 직관적인" 느낌을 Tailwind 클래스로 변환하도록 유도하고 미세 조정 |
| **DevOps** | 설정 파일 작성 | **IaC(Infrastructure as Code)** | AWS 배포 스크립트 및 GitHub Actions 워크플로우 생성을 AI에 위임하여 인프라 구축 |

### **2.3 데이터 스키마 설계 (JSON)**

서버 없이 정적 호스팅을 하기 위해 데이터를 JSON 파일로 관리한다.  
**[표 1] 제안된 JSON 데이터 스키마 구조**

| 필드명 (Field) | 타입 (Type) | 설명 (Description) | 예시 (Example) |
| :---- | :---- | :---- | :---- |
| isbn | String | 고유 식별자 (ISBN13) | "9791162241598" |
| title | String | 도서 제목 | "나 혼자만 레벨업 1" |
| buyer\_stats | Object | 구매자 분포 데이터 (핵심 추천 지표) | {"20s\_female": 35.0, "30s\_male": 15.0} |
| rating\_stats | Object | 별점별 분포 데이터 | {"5star": 80, "1star": 2} |
| metrics | Object | 집계된 지표 | {"total\_sales\_index": 50000} |

## **3 데이터 엔지니어링: 수집 및 가공 (Data Engineering)**

만화 선택의 근거가 될 데이터를 확보하는 단계이다.

### **3.1 데이터 소스 분석 및 접근 전략**

* **Target:** 알라딘 만화 카테고리 상세 페이지.  
* **Strategy:** TTB API로 기본 목록을 확보하고, Playwright로 상세 페이지에 접속하여 '구매자 분포' 텍스트와 '평점' 데이터를 파싱한다.  
* **Anti-Bot:** User-Agent 로테이션과 Random Delay를 적용하여 차단을 방지한다.

### **3.2 상세 스크레이핑 로직 (Vibe Coding 적용)**

AI에게 다음과 같은 구체적인 지시(Prompt)를 통해 코드를 생성한다:  
"Playwright를 사용하여 알라딘 상세 페이지의 '구매자 분포' 텍스트(예: 20대 여성 35%)를 찾아서, 정규표현식으로 파싱해 JSON 객체로 변환하는 NodeJS 함수를 작성해줘. 선택자는 텍스트 내용을 기반으로 찾아야 해."

### **3.3 데이터 파이프라인**

* **주기:** 주 1회 업데이트 (GitHub Actions).  
* **저장:** 가공된 데이터를 data.json으로 저장하여 프론트엔드에서 바로 로드.

## **4 프론트엔드 개발: 모바일 웹 구축 (Frontend Development)**

"그냥 아무거나 보고 싶을 때" 빠르고 쉽게 선택할 수 있는 직관적인 UI를 목표로 한다.

### **4.1 기술 스택: Astro TypeScript**

* **Astro:** 불필요한 자바스크립트를 제거하여 모바일에서 즉각적인 로딩 속도를 보장한다.9  
* **Tailwind CSS DaisyUI:** 미니멀하고 깔끔한 디자인을 빠르게 구현한다.

### **4.2 모바일 UI/UX 설계**

* **Simple Filter:** 복잡한 검색 대신 "30대 남성", "20대 여성" 탭 하나만 누르면 해당 그룹의 베스트 만화가 정렬되도록 한다.  
* **Visual Cards:** 도서 표지를 강조하고, 하단에 해당 세대의 구매 비중을 막대 그래프(Bar)로 심플하게 표시한다.  
* **Random Pick:** 사용자가 결정을 내리기 힘들 때를 대비해, 상위 랭킹 내에서 무작위로 책을 추천해주는 '랜덤 뽑기' 버튼을 구현한다.

## **5 DevOps: 인프라 및 배포 (DevOps & Deployment)**

### **5.1 AWS S3 CloudFront 정적 호스팅**

* **구조:** S3에 빌드된 정적 파일 저장, CloudFront(CDN)로 전 세계 고속 배포.  
* **보안:** OAC(Origin Access Control)를 적용하여 S3 직접 접근 차단.

### **5.2 GitHub Actions (CI/CD)**

* GitHub 이슈에 "크롤링 시작"이나 "배포" 라벨을 붙이면 자동으로 데이터 수집 스크립트가 돌거나 Astro 빌드가 수행되도록 자동화한다. 이는 바이브 코딩 워크플로우의 효율성을 높여준다.

## **6 결론 (Conclusion)**

프로젝트 **'이아고'**는 두 가지 측면에서 중요한 의미를 갖는다.

1. **실용적 가치:** 알라딘의 숨겨진 데이터를 양지로 끌어내어, 만화 독자들에게 "실패 없는 선택"과 "취향 저격"의 즐거움을 제공한다.  
2. **교육적 가치:** 기획자, 개발자, 디자이너의 역할을 AI와 함께 수행하며, **바이브 코딩**이라는 차세대 개발 방법론을 완벽하게 내재화하는 계기가 된다.

이 프로젝트를 통해 구축된 시스템은 향후 개인적인 만화 라이브러리로 확장되거나, 다른 카테고리(소설, 인문 등)로의 확장성도 기대할 수 있다.
